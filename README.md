#  Supply Chain Analytics Dashboard 

This project presents an interactive **Supply Chain Analytics Dashboard** built using **Power BI**, focused on analyzing sales performance, customer behavior, and regional market trends using real-world supply chain data.

- LATAM and Europe are the top-performing markets

- Sales trends show a noticeable decline toward the later period
